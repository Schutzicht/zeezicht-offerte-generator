# Zee-Zicht Offerte Tool

Dit is een eenvoudige tool om offertes te genereren in de Zee-Zicht huisstijl.

## Hoe werkt het?
1. Open `index.html` in je browser (Chrome, Safari, Firefox).
2. Vul aan de linkerkant alle gegevens in.
3. Rechts zie je direct het resultaat.
4. Klik op "Download PDF" om de offerte op te slaan.

## Aanpassingen
*   **Logo/Stijl:** Pas `style.css` aan.
*   **Teksten:** Pas `index.html` of `script.js` aan voor standaard waarden.
